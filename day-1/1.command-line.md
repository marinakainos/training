# Command line

### Open terminal
`cmd + space` to search programs on Mac

### Navigation
```
pwd
ls
ls -a
ls -l
cd directory_name
cd ..
cd ~
```

### Working with files
```
mkdir directory_name
rmdir directory_name
mv old_name new_name
touch file.txt
rm file.txt
rm -r directory_name
```

```
cat file1
cat >> file1
head file1
tail file1
echo "Hello to you too!" > file2
open file2

cp file1 file2
cp -r dir1 dir2

head file & tail file2
cat long_file | grep word
```

### Text editors
Open file_name in the nano editor, installed by default

`nano file_name`

Open file_name in the Vim editor

`vim file_name`

### Network utilities
Check the network connection to host

`ping host`

Download the file from url

`curl url`

### File permissions
Change the permissions of file_name to permissions (e.g., chmod 755 file_name)

`chmod permissions file_name`

### Miscellaneous
Display the manual for command.

`man command`

Print text to the terminal.

`echo "text"`

Show the current date and time.

`date`

Show the month's calendar.

`cal`

Show how long the system has been running.

`uptime`

Create an alias new that executes commands.

`alias new='commands'`

Wipe the display

`clear`
