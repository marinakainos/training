# Git

### Prerequisites
- Go to <GitHub.com> and create an account
- Download Chrome & Visual Studio Code

### Installation
Git comes pre installed on MacBooks, you would only have to activate it.
To confirm it's ready, check which version is installed using the following command: `git version`. You might get a pop up asking to install XCode - accept that.

### Configuration
```
git config list
git config --global user.name "Your Name"
git config --global user.email "your.email@example.com"
```

### Creating a local repo
Create a repository in GitHub

```
echo "# day1-test" >> README.md
git init
git add README.md
git commit -m "first commit"
git branch -M main
git remote add origin git@github.com:marinakainos/day1-test.git
git push -u origin main
```

```
git help
git status
git log
git log --oneline
git diff
git diff --staged

git restore --staged file.txt
git commit --amend -m "New commit message"

git stash
git stash apply/pop
```
